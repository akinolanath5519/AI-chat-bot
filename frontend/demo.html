Experience the chatbot interface without needing a backend server

About This Demo

This is a demonstration of the AI chatbot interface. It simulates the behavior of the chatbot without requiring a backend server or API keys. This allows you to test the appearance and functionality of the chat widget before setting up the full system.

Click on the chat icon in the bottom right corner to start interacting with the demo chatbot.

Features Demonstrated

Chat Interface

Experience the responsive chat interface with message bubbles, typing indicators, and smooth animations.

Lead Collection

See how the lead collection form appears when triggered by specific messages in the conversation.

Responsive Design

Test how the chatbot adapts to different screen sizes by resizing your browser window.

Next Steps

After exploring this demo, you can set up the full version by:

  1. Configuring your Claude API key
  2. Setting up Google Sheets integration for lead storage
  3. Configuring email notifications
  4. Starting the backend server
  5. Integrating the chatbot into your website

Refer to the documentation files in the project for detailed instructions on each step.
