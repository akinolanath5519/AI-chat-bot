Website with AI Chatbot Integration

This is a demo page showing how the AI chatbot can be integrated into any website.

Click on the chat icon in the bottom right corner to start a conversation with the AI assistant.

How It Works

The chatbot uses Anthropic's Claude to provide intelligent responses to user queries. It can:

  • Answer questions about products and services
  • Provide customer support
  • Collect lead information when appropriate
  • Maintain context throughout the conversation

Integration Instructions

To integrate this chatbot into your own website:

  1. Include the chatbot.js script in your HTML
  2. Add your API endpoint in the configuration
  3. Customize the appearance to match your brand
<script src="path/to/chatbot.js"></script>
<script>
  initChatbot({
    apiUrl: 'https://your-api-endpoint.com',
    theme: 'light'
  });
</script>
